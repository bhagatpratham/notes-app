# NOTES CLI APP

Manage notes in a single JSON file 

### Features

1. Create note
2. Remove note
3. List notes
4. Read note

### Get Started

1. Clone this repo 
2. In your command line 

```
cd notes-app
```

```
npm install
```

### Commands

Add Note

```
node index add --title <title> --body <body>
```

Remove Note

```
node index remove --title <title>
```
